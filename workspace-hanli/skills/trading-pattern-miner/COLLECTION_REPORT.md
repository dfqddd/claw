## 📌 技术说明

### 数据收集方式
1. **浏览器自动化** - 用于访问页面、提取动态内容
2. **Python 脚本** - 用于批量处理、数据清洗
3. **Cookie 登录** - 保持登录状态，避免验证码

### 遇到的挑战
1. **反爬机制** - 需要 Cookie 登录
2. **动态加载** - 部分内容需要 JavaScript 渲染
3. **HTML 结构复杂** - 需要精确的 selector

### 解决方案
1. ✅ 使用浏览器工具直接访问
2. ✅ 用 JavaScript 提取数据
3. ✅ 批量保存为 JSON 格式
