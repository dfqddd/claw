## 🎯 STRATEGY-001: 龙回头

**模式来源**: 炒股养家、作手新一  
**策略类型**: 龙头战法  
**风险等级**: 高  
**适合行情**: 主线明确的上涨/震荡行情

### 策略逻辑

```
市场出现明确主线 → 龙头股连续涨停 (3 板+) → 第一次分歧开板 → 
龙回头确认 (不跌破 10 日线) → 第二波主升浪
```

### 买入条件 (必须同时满足 5 条)

| 条件 | 要求 | 验证方法 |
|------|------|---------|
| **1. 板块地位** | 必须是板块龙头 | 同板块涨停时间最早/连板高度最高 |
| **2. 连板数量** | 至少 3 连板 | 查看 F10 涨停统计 |
| **3. 第一次分歧** | 首次开板或断板 | 之前未开过板 |
| **4. 量能配合** | 分歧日成交量>前一日 1.5 倍 | 看成交量柱 |
| **5. 承接有力** | 分歧后 3 日内不跌破 10 日线 | 看 K 线位置 |

### 买入时机

**最佳买点**: 分歧后第 2-3 天，回踩 10 日线附近

**仓位**:
```
- 首次买入：30% 仓位 (试探)
- 确认走强：加仓至 70% (突破分歧日高点)
- 主升浪：满仓 (连续涨停)
```

### 卖出条件 (满足任一即卖出)

| 条件 | 操作 | 说明 |
|------|------|------|
| **涨幅>30%** | 减仓 50% | 锁定利润 |
| **跌破 5 日线** | 清仓 | 趋势转弱 |
| **跌破买入价 -8%** | 止损清仓 | 严格止损 |
| **持仓>15 天** | 清仓 | 时间止损 |

### 仓位管理

```python
# 单只股票最大仓位：100%
# 同时持有股票数：1-2 只
# 总仓位控制：
#   - 主线明确期：80-100%
#   - 震荡期：50-70%
#   - 退潮期：0-20%
```

### 选股公式 (通达信)

```
{龙回头选股公式}
连板数:=BARSLASTCOUNT(C/REF(C,1)>1.095);
首次分歧:=REF(连板数，1)>=3 AND C/REF(C,1)<1.095;
回踩确认:=L<=MA(C,10)*1.02 AND C>MA(C,10);
成交量:=VOL>REF(MA(VOL,5),1)*1.5;

XG: 首次分歧 AND 回踩确认 AND 成交量;
```

### 实战案例

**案例**: 豫能控股 (2026-03)  
**操作者**: 玫瑰超短

**过程**:
- 年前 11 号启动，27 号 5 连板
- 涨价消息扰乱，3.2 号分歧大阳线
- 3.3 号炸板深水，但竞价超预期 0 轴开
- 资金疯抢快速拉升，符合龙回头

**结果**: 持股至今 25%+利润

## 🎯 STRATEGY-002: 龙头首阴

**模式来源**: 龙飞虎、赵老哥  
**策略类型**: 龙头战法  
**风险等级**: 极高  
**适合行情**: 情绪高涨期

### 策略逻辑

```
龙头股连续涨停 → 首次收阴线 (非跌停) → 次日反包预期 → 买入
```

### 买入条件

| 条件 | 要求 |
|------|------|
| **1. 龙头地位** | 市场公认龙头 (连板高度最高) |
| **2. 连板数** | 5 板以上为佳 |
| **3. 首根阴线** | 第一次收阴 (之前全是阳线/涨停) |
| **4. 非跌停** | 收盘价>跌停价 (有承接) |
| **5. 量能** | 成交量<前一日 2 倍 (未失控) |

### 买入时机

**最佳买点**: 首阴次日，竞价高开或平开

**仓位**:
```
- 竞价高开：50% 仓位
- 竞价平开：30% 仓位
- 竞价低开：观望 (不符合预期)
```

### 卖出条件

| 情况 | 操作 |
|------|------|
| **次日涨停** | 持有，设好止盈位 |
| **次日大涨>5%** | 减仓 50% |
| **次日平盘** | 持有观察 |
| **次日低开低走** | 立即止损 |
| **跌破首阴最低价** | 止损清仓 |

### 止损规则

```
- 固定止损：-8%
- 时间止损：买入后 3 天不涨即走
- 技术止损：跌破首阴最低价
```

## 🎯 STRATEGY-003: 20 日线趋势

**模式来源**: 星辰趋势主升、二池  
**策略类型**: 趋势跟踪  
**风险等级**: 中  
**适合行情**: 震荡上行行情

### 策略逻辑

```
股价站上 20 日线 → 回踩 20 日线不破 → 放量突破平台 → 主升浪
```

### 买入条件

| 条件 | 要求 | 权重 |
|------|------|------|
| **1. 均线多头** | 5 日>10 日>20 日线 | 必须 |
| **2. 回踩确认** | 回踩 20 日线不破 (3 次以内) | 必须 |
| **3. 放量突破** | 突破平台时量>5 日均量 1.5 倍 | 必须 |
| **4. MACD 金叉** | DIF 上穿 DEA | 加分项 |
| **5. 板块共振** | 所属板块指数同步走强 | 加分项 |

### 买入时机

**买点 A**: 回踩 20 日线企稳 (30% 仓位)  
**买点 B**: 放量突破平台 (加仓 50%)  
**买点 C**: 主升浪加速 (加仓 20%)

### 卖出条件

| 信号 | 操作 |
|------|------|
| **收盘价跌破 20 日线** | 清仓 |
| **MACD 死叉** | 减仓 50% |
| **短期涨幅>50%** | 减仓 50% |
| **跌破 10 日线** | 减仓 30% |

### 仓位管理

```
建仓流程:
  回踩 20 日线 → 30% 仓位
  突破平台确认 → +50% 仓位 (总 80%)
  主升浪加速 → +20% 仓位 (总 100%)
  
减仓流程:
  跌破 10 日线 → -30% 仓位
  跌破 20 日线 → 清仓
```

### 选股公式

```
{20 日线趋势选股}
均线多头:=MA(C,5)>MA(C,10) AND MA(C,10)>MA(C,20);
回踩 20 日:=L<=MA(C,20)*1.02 AND C>MA(C,20);
放量:=VOL>MA(VOL,5)*1.5;
平台突破:=C>HHV(REF(H,20),20);

XG: 均线多头 AND 回踩 20 日 AND 放量;
```

## 🎯 STRATEGY-004: 首板挖掘

**模式来源**: 职业炒手、陈药师  
**策略类型**: 打板  
**风险等级**: 高  
**适合行情**: 消息驱动行情

### 策略逻辑

```
突发利好消息 → 板块集体异动 → 最先涨停股 → 次日溢价
```

### 买入条件

| 条件 | 要求 |
|------|------|
| **1. 消息催化** | 突发利好 (政策/业绩/事件) |
| **2. 板块效应** | 板块内 3 只以上涨停 |
| **3. 时间优先** | 9:35 前涨停最佳 |
| **4. 封单强度** | 封单金额>成交金额 30% |
| **5. 市值适中** | 50-200 亿最佳 |

### 买入时机

**打板买入**: 即将封板瞬间 (9:35 前)

**仓位**:
```
- 单只股票：不超过 20% 仓位
- 同时持有：3-5 只分散风险
```

### 卖出条件

| 情况 | 操作 |
|------|------|
| **次日涨停** | 持有 |
| **次日高开>3%** | 持有 30 分钟，不涨停卖出 |
| **次日平开** | 开盘即卖出 |
| **次日低开** | 立即卖出 |
| **低开<-3%** | 立即止损 |

### 止损规则

```
- 固定止损：-5%
- 时间止损：次日不涨即走
- 不抱幻想：首板就是套利，不恋战
```

## 🎯 STRATEGY-005: 超跌反弹

**模式来源**: 小土堆爆金币、jl 韭菜抄家  
**策略类型**: 反弹  
**风险等级**: 中  
**适合行情**: 超跌行情/震荡市

### 策略逻辑

```
连续下跌 → 超卖信号 → 缩量企稳 → 阳线确认 → 反弹
```

### 买入条件

| 条件 | 要求 |
|------|------|
| **1. 跌幅够深** | 5 日跌幅>15% 或 10 日跌幅>25% |
| **2. 超卖信号** | RSI<30 或 KDJ 的 J<0 |
| **3. 缩量企稳** | 成交量缩至前期 1/3 以下 |
| **4. 阳线确认** | 出现中阳线 (涨幅>3%) |
| **5. 不再新低** | 阳线后 3 日不创新低 |

### 买入时机

**买点**: 阳线次日，确认不创新低

**仓位**:
```
- 首根阳线：20% 仓位 (试探)
- 突破 5 日线：+30% 仓位 (总 50%)
- 接近 10 日线：-30% 仓位
- 破位：清仓
```

### 卖出条件

| 信号 | 操作 |
|------|------|
| **反弹至 10 日线** | 减仓 50% |
| **反弹至 20 日线** | 清仓 |
| **跌破阳线最低价** | 止损清仓 |
| **持仓>10 天** | 清仓 |

## ⚠️ 风险控制

### 止损纪律

```
铁律 1: 任何策略，亏损达到 -8% 必须止损
铁律 2: 连续止损 3 次，停止交易 1 周
铁律 3: 单月亏损>15%，停止交易 1 个月
铁律 4: 不退潮期不做多，不逆势操作
```

### 常见错误

| 错误 | 后果 | 避免方法 |
|------|------|---------|
| 频繁换策略 | 哪种都不精 | 专注 1-2 种策略 |
| 不止损 | 小亏变大亏 | 设置自动止损单 |
| 仓位过重 | 一次失误重伤 | 严格执行仓位管理 |
| 情绪化交易 | 亏损后想翻本 | 连续止损后休息 |
| 临盘改变计划 | 容易失误 | 盘前做好计划 |

## 📈 绩效评估

### 评估指标

| 指标 | 目标值 | 计算方法 |
|------|--------|---------|
| **年化收益** | >50% | (期末资金 - 期初资金)/期初资金 |
| **最大回撤** | <-15% | 最大亏损幅度 |
| **胜率** | >50% | 盈利次数/总交易次数 |
| **盈亏比** | >2.5 | 平均盈利/平均亏损 |
| **夏普比率** | >1.5 | (年化收益 - 无风险利率)/波动率 |

### 评估周期

```
- 每日：记录交易日志
- 每周：统计盈亏，复盘策略
- 每月：评估策略有效性，调整参数
- 每季：全面复盘，优化策略组合
```
