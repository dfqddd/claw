# A股中短线分析框架

## 1. 大盘环境判断标准

### 量能
- **火热**：>1.5万亿
- **活跃**：>1万亿
- **一般**：>8000亿
- **低迷**：<8000亿

### 涨跌比
- **普涨**：>2
- **偏强**：1-2
- **偏弱**：0.5-1
- **普跌**：<0.5

### 涨停数
- **强势**：>80
- **正常**：50-80
- **冷清**：<30

## 2. 板块轮动分析

### 主线判断
- **主线**：连续3天领涨
- **一日游**：仅1天领涨

### 强势板块
- 板块内涨停数>5

### 龙头股辨识
- 最先涨停
- 最高连板

## 3. 资金面解读

### 北向资金
- **强流入**：单日>50亿
- **大流出**：单日<-50亿
- **趋势判断**：连续3日同向流动 = 趋势确认

### 龙虎榜（必查项）

#### 数据来源
- 数据库字段：`capital_flow.dragon_tiger_json`
- 每条记录包含：code, name, change_pct, reason, buy_yi, sell_yi, net_yi, lhb_reason

#### 分析维度
1. **机构动向**：
   - 机构净买入 >1亿 = 看多信号
   - 机构净卖出 >1亿 = 看空信号
   - 多家机构同向 = 信号更强
2. **行业方向**：
   - 统计上榜个股所属行业，找出资金集中方向
   - 与当日板块排名交叉验证
3. **持仓关联**（重要）：
   - 检查持仓股是否上榜（直接风险/机会信号）
   - 检查持仓股同行业个股是否上榜（间接信号）
   - 同行业被机构卖出 = 板块偏空，持仓股反弹动力不足
   - 同行业被机构买入 = 板块偏多，持仓股有望跟涨
4. **净买入排名**：
   - 净买入前5 = 当日最强聪明钱方向
   - 净卖出前5 = 当日最危险方向

#### 判断标准
| 信号 | 条件 | 含义 |
|------|------|------|
| 🔥 强烈看多 | 机构净买入 >5亿 | 主力重仓进场 |
| ✅ 偏多 | 机构净买入 1-5亿 | 机构认可 |
| ⚠️ 偏空 | 机构净卖出 1-5亿 | 机构减仓 |
| 🚨 强烈看空 | 机构净卖出 >5亿 | 主力出逃 |
| ⚡ 涨停但净卖出 | 涨停 + 净卖出 | 借涨停出货，高度警惕 |

### 融资余额
- 连续增加 = 市场看多
- 连续减少 = 市场看空
- 单日变化 >50亿 = 异常信号

## 4. 市场情绪周期

### 阶段特征
- **冰点**：连板高度≤2，封板率<40%
- **修复**：连板高度提升，封板率改善
- **发酵**：题材扩散，赚钱效应增强
- **高潮**：连板≥5，封板率>70%
- **退潮**：高位股跌停，亏钱效应显现

### 操作策略
- 冰点是低吸机会
- 高潮后注意风险

## 5. 量价分析框架

### 竞价阶段
- **高开放量**：强势
- **高开缩量**：谨慎

### 盘中
- **量增价涨**：看多
- **量增价跌**：看空

### 量比指标
- **>2**：异动
- **<0.5**：极度缩量

## 6. 近3天趋势对比方法

### 量能趋势
- **连续放量**：资金进场
- **连续缩量**：观望

### 情绪趋势
- **连板高度递增**：升温
- **连板高度递减**：降温

### 板块趋势
- **同一板块连续领涨**：持续性强
