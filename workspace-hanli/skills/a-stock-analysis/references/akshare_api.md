# AKShare 常用 API 速查

## 指数数据

### stock_zh_index_spot_em
- **用途**：东方财富指数实时行情
- **关键参数**：无
- **返回字段**：代码、名称、最新价、涨跌额、涨跌幅、成交量、成交额

### stock_zh_index_daily_em
- **用途**：东方财富指数历史行情
- **关键参数**：symbol（指数代码）、start_date、end_date
- **返回字段**：日期、开盘、收盘、最高、最低、成交量、成交额、涨跌幅

## 个股行情

### stock_zh_a_spot_em
- **用途**：东方财富A股实时行情
- **关键参数**：无
- **返回字段**：代码、名称、最新价、涨跌幅、成交量、成交额、换手率、市盈率

### stock_zh_a_hist
- **用途**：A股历史行情
- **关键参数**：symbol（股票代码）、period（日线/周线/月线）、start_date、end_date、adjust（前复权/后复权）
- **返回字段**：日期、开盘、收盘、最高、最低、成交量、成交额、涨跌幅、换手率

## 板块数据

### stock_board_concept_name_em

- **关键参数**：无
- **返回字段**：板块名称、最新价、涨跌幅、涨跌额、成交量、成交额

### stock_board_industry_hist_em
- **用途**：行业板块历史行情
- **关键参数**：symbol（板块名称）、start_date、end_date
- **返回字段**：日期、板块名称、开盘、收盘、最高、最低、成交量、成交额、涨跌幅

### stock_board_concept_hist_em
- **用途**：概念板块历史行情
- **关键参数**：symbol（板块名称）、start_date、end_date
- **返回字段**：日期、板块名称、开盘、收盘、最高、最低、成交量、成交额、涨跌幅

## 资金流向

### stock_hsgt_north_net_flow_in_em
- **用途**：北向资金净流入
- **关键参数**：无
- **返回字段**：日期、北向资金净流入、沪股通净流入、深股通净流入

### stock_individual_fund_flow_rank
- **用途**：个股资金流向排行
- **关键参数**：date（日期）
- **返回字段**：代码、名称、最新价、涨跌幅、主力净流入、超大单净流入、大单净流入、中单净流入、小单净流入

### stock_hsgt_hist_em
- **用途**：沪深港通历史数据
- **关键参数**：symbol（股票代码）、indicator（持股数量/持股比例）
- **返回字段**：日期、股票代码、持股数量、持股比例

## 龙虎榜

### stock_lhb_detail_em
- **用途**：龙虎榜详情
- **关键参数**：date（日期）
- **返回字段**：代码、名称、收盘价、涨跌幅、龙虎榜净买入、买入席位、卖出席位、营业部名称

## 涨停/跌停池

### stock_zt_pool_em
- **用途**：涨停股票池
- **关键参数**：date（日期）
- **返回字段**：代码、名称、最新价、涨跌幅、首次涨停时间、封单金额、涨停原因

### stock_zt_pool_dtgc_em
- **用途**：打开涨停股票池
- **关键参数**：date（日期）
- **返回字段**：代码、名称、最新价、涨跌幅、涨停时间、打开时间、炸板次数

### stock_zt_pool_zbgc_em
- **用途**：涨停失败股票池
- **关键参数**：date（日期）
- **返回字段**：代码、名称、最新价、涨跌幅、最高涨停价、跌停价、炸板原因

### stock_zt_pool_previous_em
- **用途**：昨日涨停股票池
- **关键参数**：date（日期）
- **返回字段**：代码、名称、最新价、涨跌幅、昨日涨停价、连板数

## 新闻

### stock_info_global_cls
- **用途**：全球财经新闻
- **关键参数**：symbol（股票代码，可选）、date（日期）
- **返回字段**：发布时间、新闻标题、新闻内容、相关股票

## 融资融券

### stock_margin_sse
- **用途**：上交所融资融券数据
- **关键参数**：symbol（股票代码）、start_date、end_date
- **返回字段**：日期、融资买入额、融资余额、融券卖出量、融券余额

### stock_margin_szse
- **用途**：深交所融资融券数据
- **关键参数**：symbol（股票代码）、start_date、end_date
- **返回字段**：日期、融资买入额、融资余额、融券卖出量、融券余额
