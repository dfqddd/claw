> **所属平台**：OpenClaw AI 助手
> **一句话介绍**：让 AI 像一个勤勉的研究员一样，每天自动采集市场数据、分析盘面、推送研判结论到你的钉钉群。

### 🤔 起因：一个散户的痛点

每天收盘后，你是不是也有这样的困扰？

- 打开十几个网页，翻看大盘指数、涨跌家数、板块排名、涨停数据……
- 手动记录成交额变化趋势，判断市场情绪是冰点还是高潮
- 想做一个完整的复盘，但信息太分散，等整理完已经快睡觉了

**我们想做的事情很简单：把这些重复性的"苦力活"交给 AI。**

不是让 AI 帮你炒股（它也不会），而是让它帮你**看盘、整理数据、生成分析报告**——就像你身边有一个 7×24 小时在线的研究助理。

---

### 🏗️ 设计理念：AI 不是黑盒，而是你的"数据管家"

很多人对 AI 在金融领域的应用有一个误解：觉得 AI 就是一个"预测涨跌的黑盒"。但我们的设计理念完全不同——

#### 核心原则：**AI 负责采集和整理，人负责决策**

我们把整个系统拆成了三层：

```
┌─────────────────────────────────────────────┐
│           OpenClaw AI 助手（对话层）           │
│     "帮我分析今天的盘面" → 自然语言交互         │
├─────────────────────────────────────────────┤
│           智能分析引擎（分析层）                │
│   7 维度 Checklist × 动态分位数 × 趋势对比     │
├─────────────────────────────────────────────┤
│           自动化数据中台（数据层）              │
│   多数据源采集 → SQLite 缓存 → 完整性保障      │
└─────────────────────────────────────────────┘
```

**数据层**不依赖任何 AI 模型，它是纯工程化的数据采集和存储系统，保证数据的准确性和完整性。

**分析层**用规则引擎做量化分析，不是"AI 拍脑袋"，而是基于明确的指标体系和历史分位数做客观判断。

**对话层**才是 AI 真正发挥作用的地方——通过 OpenClaw 的大模型能力，把冰冷的数据转化为自然语言的分析报告，让你像跟研究员聊天一样获取市场信息。

### 🧠 AI 创作理念：三个关键词

#### 1️⃣ **结构化**：7 维度 Checklist，不遗漏

我们设计了一套严格的 **7 维度分析框架**，确保每次分析都是完整的、不遗漏的：

| 维度 | 核心指标 | 判断方式 |
|------|---------|---------|
| 🏛️ 大盘环境 | 三大指数、成交额、涨跌家数 | 动态分位数 + 固定阈值 |
| 🔄 板块轮动 | 行业/概念排名、主线识别 | 连续性判断 |
| 💰 资金面 | 北向资金、龙虎榜、主力资金 | 净流入/流出趋势 |
| 🎭 市场情绪 | 涨停数、连板高度、封板率 | 情绪周期模型 |
| 📰 消息面 | 政策催化、个股新闻 | 关键词过滤 |
| 📈 技术面 | K 线、均线、量价关系 | 技术指标计算 |
| 🎯 综合研判 | 汇总以上 6 个维度 | 多因子加权 |

这不是 AI 随意发挥，而是**有章可循的标准化分析流程**。AI 的角色是在这个框架下，把数据"翻译"成人话。

#### 2️⃣ **客观化**：动态分位数，不拍脑袋

"今天成交额 1.2 万亿，算多还是算少？"

这个问题没有绝对答案。牛市里 1.2 万亿可能算缩量，熊市里可能算放量。所以我们采用了**动态分位数**的判断方式：

- 取最近 **20 个交易日**的数据分布
- 计算 P25 / P75 / P90 分位数
- 当前值落在哪个区间，就给出对应的判断

```
极度萎缩 ← P25 → 偏弱 ← P50 → 正常 ← P75 → 偏强 ← P90 → 极度亢奋
```

同时保留**绝对参考线**作为兜底：
- 成交额 < 6000 亿 → 极度萎缩（无论分位数如何）
- 成交额 > 2 万亿 → 极度亢奋

这样既能适应不同市场环境，又不会在极端情况下失灵。

#### 3️⃣ **自动化**：全天候运转，不偷懒

整个系统像一台精密的机器，每天自动运转：

```
08:30  📥 盘前数据同步（昨日收盘数据补全）
08:45  📤 盘前预判推送（基于昨日数据的开盘展望）
15:30  📥 盘后数据同步（当日完整数据入库）
15:35  📤 收盘复盘推送（当日完整盘面分析）
18:30  📤 晚间深度推送（含龙虎榜等深度数据）
22:00  🔍 数据完整性检查 + 自动补偿
```

如果某次同步失败了？系统会在 22:00 的补偿任务中自动发现并重新同步。如果补偿也失败了？会发送钉钉告警通知你。

### 🔧 核心技术亮点

#### 多数据源智能切换

A 股数据源是出了名的不稳定——接口限流、数据延迟、格式变更是家常便饭。我们设计了一套**数据源管理器**：

```
数据源 A（优先级 0）→ 失败 → 自动切换
数据源 B（优先级 1）→ 失败 → 继续降级
数据源 C（优先级 2）→ 成功 → 返回数据
```

每个数据源都有**健康度监控**：最近 10 次请求的成功率低于 80%，自动标记为降级状态，优先使用其他数据源。这样即使某个接口挂了，系统也能继续正常运转。

#### 情绪周期模型

市场情绪不是简单的"好"或"坏"，它有自己的周期：

```
冰点 → 修复 → 发酵 → 高潮 → 退潮 → 冰点
```

我们通过**多空指数**（涨跌比）和**恐惧贪婪指数**（综合涨停数、封板率、换手率等）来量化情绪，并判断当前处于周期的哪个阶段。这对于把握市场节奏非常有帮助。

#### 与 OpenClaw 的深度集成

作为 OpenClaw 的一个 Skill 模块，这个技能可以被 AI 助手自然地调用：

```
你：今天大盘怎么样？
AI：[自动调用盘面分析] 今天三大指数全线收涨，沪指涨 0.8%，
    成交额 1.15 万亿处于近 20 日 P65 分位，属于正常偏强水平。
    涨跌比 2.3:1，市场呈现普涨格局。情绪面上，涨停 42 家，
    连板高度 5 板，封板率 72%，处于发酵阶段...

你：哪些板块比较强？
AI：[自动调用板块分析] 今天领涨的是半导体和人工智能板块，
    其中半导体已经连续 3 天领涨，有主线迹象...
```

AI 不是在"编故事"，它背后是**真实的数据查询和规则化分析**，每一个数字都有据可查。

### 💭 写在最后

这个项目最让我们自豪的，不是用了多么高深的 AI 技术，而是**找到了 AI 真正能发挥价值的场景**。

AI 不擅长预测股价（没有人擅长），但它擅长：
- **不知疲倦地采集数据**——每天定时同步，从不偷懒
- **客观地整理和分析**——不带情绪，不会因为亏钱就看空
- **用人话表达结论**——把枯燥的数字变成可读的分析报告

这就是我们的 AI 创作理念：**不追求 AI 的"神奇"，而是追求 AI 的"靠谱"。**

让 AI 做它擅长的事，让人做人擅长的事。AI 帮你看盘，你来做决策。
